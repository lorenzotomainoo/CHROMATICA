# CHROMATICA

CHROMATICA is a modern web application for generating and managing color palettes. It is designed for designers, developers, and creatives who need a fast, intuitive, and visually clean tool to explore and create color combinations.

The project is built with a simple multi-page structure, separating core functionalities such as palette generation, navigation, and user registration.

---

## Features

- Instant color palette generation  
- Modern and clean UI design  
- Fully responsive layout for desktop and mobile devices  
- Multi-page structure for better organization  
- Interactive navigation system  
- Modular and scalable codebase  

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Font Awesome (icons)  

---


---

## Pages

### Home (`index.html`)
Landing page that introduces the project and provides access to the main features.

### Palette Generator (`palette.html`)
Main tool of the application used to generate, explore, and manage color palettes.

### Registration (`registrazione.html`)
Page dedicated to user registration and authentication.

## Installation

Clone the repository:

```bash
git clone https://github.com/lorenzotomainoo/CHROMATICA.git
```

```bash
cd CHROMATICA
```
open index.html in your browser.
